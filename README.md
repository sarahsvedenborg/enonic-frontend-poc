# Redd Cross PoC monorepo

## Features

- **Frontend for Enonic content**
- **Frontend for Sanity content**
- **Sanity studio**

### Frontends

- **Next.js 14**: Latest version with App Router
- **PostCSS**: Custom CSS with PostCSS processing

### Sanity studio
- hosted on https://rk-poc.sanity.studio/

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. **Install dependencies**:
   ```bash
   npm install
   # or
   yarn install
   ```

2. **Run development server**:
   ```bash
   npm run dev
   # or
   yarn dev
   ```

3. **Open your browser**:
   Navigate to [http://localhost:3000](http://localhost:3000)

## Available Scripts

- `npm run dev:enonic` - Start development server for enonic frontend
- `npm run dev:sf` - Start development server for sanity frontend
- `npm run dev:sanity` - Start development server for sanity studio (runs on localhost:3333)
- see package.json for other scirpts
